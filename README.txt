Hi Ultimate Mobile V8

주요 기능
- 오늘의 관심도 TOP 3
- 한국·미국 TOP 10
- 주요 뉴스와 급등 감지
- 뉴스 영향도
- 관련 대장주 6개
- 관심종목 저장
- 검색과 국가별 필터
- 실시간 뉴스·현재가 연결
- 연결 실패 시 예시 데이터 자동 표시

설치
1. ZIP 압축을 풉니다.
2. GitHub 저장소의 index.html, manifest.json, sw.js, icon.svg를 교체합니다.
3. Commit changes를 누릅니다.
4. 1~3분 뒤 GitHub Pages 주소를 새로고침합니다.

중요
- 실제 AI 서버가 수백 건의 뉴스를 분석하는 구조는 아닙니다.
- 브라우저에서 뉴스 제목과 가격 변동률을 단순 분석합니다.
- 자동매매, 실제 주문, 수익 보장 기능은 없습니다.
